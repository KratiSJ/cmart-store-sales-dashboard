# 📊  CMart Store Dashboard

## 🔍 Project Overview

This project presents a comprehensive sales analysis dashboard built using **Google Looker Studio**. It provides insights into sales performance, product categories, outlet types, and customer trends.

---

## 📁 Dataset

- Source: Internal dataset (attached in repository)
- File: `CMart Stores Data.xlsx`
- Contains:
  - Product details
  - Sales values
  - Outlet types
  - Location data
