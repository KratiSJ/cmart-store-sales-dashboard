# 🛒 CMART Store Dashboard

## 📌 Project Overview
The **CMART Store Dashboard** is an interactive Business Intelligence dashboard developed using **Google Looker Studio** to analyze store performance and business trends.

This dashboard provides detailed insights into:
- Sales Performance
- Profit Analysis
- Customer Distribution
- Regional Performance
- Product Categories
- Monthly Trends

The project helps businesses make data-driven decisions through visually appealing and interactive reports.

---

# 🚀 Dashboard Features

✅ Interactive KPI Cards  
✅ Dynamic Filters & Slicers  
✅ Regional Sales Analysis  
✅ Customer Segmentation  
✅ Profit & Sales Trend Analysis  
✅ Country-wise Performance Tracking  
✅ Map-based Visualization  
✅ Search Functionality for Customers
